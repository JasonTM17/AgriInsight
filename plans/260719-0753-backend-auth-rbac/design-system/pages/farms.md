# Farms page override

Primary job: compare farms, fields, and seasons, then drill to activity and harvest lineage.

- Composition: map/plot context paired with a sortable farm-season table; filters remain visible and URL-addressable.
- Show hectares, yield, cost/ha, target variance, season status, and the evidence/run link together.
- Empty state explains how to provision a farm; denied state never reveals another tenant's names or counts.
- Mobile: map can collapse behind a labelled “View plot” sheet; table falls back to priority rows without hidden actions.
